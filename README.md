## Projetos

### [Flecto](https://github.com/ggabriell07/flecto)
Assistente financeiro comportamental operando via WhatsApp. Sistema multimodal em produção — processa texto, áudio e imagem, orquestra LLMs para análise comportamental e persiste dados com histórico em PostgreSQL.

`n8n` `Claude API` `OpenAI Whisper` `Evolution API` `Supabase` `HKDF` `JSONB`

---

### [HermesHub](https://github.com/ggabriell07/hermeshub)
SaaS de criação de conteúdo para Instagram com geração por LLM. Integração completa com Instagram Graph API via OAuth2, token AES-GCM-256 encriptado, frontend React com wizard de onboarding em 4 etapas.

`React` `TypeScript` `Supabase` `Claude Sonnet` `Instagram Graph API` `AES-GCM-256` `GitHub Actions`

---

### CondoClaro *(em desenvolvimento)*
Ferramenta de leitura e tradução de balancetes financeiros de condomínios via Claude Vision. Upload de PDF → relatório em linguagem acessível em menos de 60 segundos.

`Claude Vision` `n8n` `Supabase` `PDF parsing`
